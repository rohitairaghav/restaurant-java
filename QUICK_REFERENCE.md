# Quick Reference Guide

A one-page quick reference for common tasks in the Restaurant Inventory Management System.

## Login
- **URL**: Your restaurant's inventory URL
- **Credentials**: Provided by manager
- **Forgot Password**: Contact manager

## Navigation Shortcuts

| Section | What You Can Do |
|---------|----------------|
| **Dashboard** | View overview, recent activity |
| **Inventory** | Add/edit/delete items, set thresholds |
| **Stock** | Record transactions, view history |
| **Suppliers** | Manage supplier information (Managers only) |
| **Recipes** | Create recipes with ingredients |
| **Sales** | Record sales, update inventory |
| **Alerts** | View low stock warnings |
| **Analytics** | View usage reports |
| **Reports** | Generate and export reports |
| **Users** | Manage user accounts (Managers only) |

## Common Tasks

### Add Inventory Item
1. Inventory → Add Item
2. Fill: Name, Category, Unit, Cost, Stock, Threshold
3. Save

### Record Stock Transaction
1. Stock → Add Transaction
2. Select: Item, Type (In/Out), Quantity, Reason
3. Save

### Record Sale
1. Sales → Record Sale
2. Select: Recipe, Quantity
3. Check "Update Inventory" to auto-deduct ingredients
4. Save

### View Alerts
1. Click Alerts in navigation (badge shows count)
2. View low stock and out-of-stock items
3. Mark as read after addressing

### Create Recipe
1. Recipes → Add Recipe
2. Fill: Name, Quantity, Unit
3. Add ingredients with quantities
4. Save

## Role Permissions

### Manager ✅
- Full access to all features
- Can manage users
- Can delete items
- Can manage suppliers
- Can view analytics

### Staff ✅
- Can add/update inventory
- Can record transactions
- Can create recipes
- Can record sales
- ❌ Cannot delete items
- ❌ Cannot manage suppliers
- ❌ Cannot manage users
- ❌ Cannot edit transactions older than 24 hours

## Keyboard Shortcuts
- `Ctrl/Cmd + K`: Quick search
- `Esc`: Close modal
- `Enter`: Submit form

## Offline Mode
- Works without internet
- Data syncs automatically when online
- Check connection indicator (top-right)
